## About this code  
The Geant4 code is a simulation about scintillator.

### Geometry  
1) World  
2) Detector (just for checking a shape of beam)  

### Source   
1) Phase space file (filename: Phsp_Iplan_1_2_1_v2.txt)   

### Scoring    
1) Detector (Voxel geometry 100 x 100)
